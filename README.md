# Ws-NumExtract — WhatsApp Chat Number Extractor

Extract phone numbers from WhatsApp Web chats and export them.

---

## 📦 Installation

1. Open `chrome://extensions/`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select `Ws-NumExtract` folder

---

## 🚀 Usage

1. Open https://web.whatsapp.com
2. Open a chat
3. Click extension icon
4. Choose:
   - ⚡ Quick Mode
   - 🔍 Deep Scan Mode
5. Click **EXTRACT NUMBERS**
6. View results

---

## 📤 Download Options

- **TXT** → one number per line
- **CSV** → spreadsheet format
- **JSON** → structured data

---

## 📥 From GitHub Release

1. Go to Releases page
2. Download latest `Ws-NumExtract.zip`
3. Extract the ZIP file
4. Go to `chrome://extensions/`
5. Enable Developer Mode
6. Click **Load unpacked**
7. Select extracted folder

---

## ⚙️ Notes

- Works only on WhatsApp Web
- Group chats excluded
- Data stays local
